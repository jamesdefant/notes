GitHub Desktop

GitHub Desktop is a version control system that harnesses the power of Git and filters out the complexity of working with the command line interface (CLI).

The workflow to use GitHub Desktop looks like this:

  1. Clone/Create/Add a repository (setup your local repo)
  2. Define which Branch you're using (if you're using more than one Branch)
  3. Write a summary note and Commit to your local repo
